## Usage:
### plug and play
```bash
bash <(curl -Ls https://github.com/xinyangyuan/create/raw/master/create.sh)
```
### install locally
- install
```bash
git clone "https://github.com/xinyangyuan/create.git"
cd create
source ~/create.sh
```
- call create to use it
```bash
create <name of your project>
```
